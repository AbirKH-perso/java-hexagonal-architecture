# kata-bank-account

## Requirements

For building and running the application you need:

- [JDK 1.19](https://www.oracle.com/java/technologies/downloads/#java19)
- [Maven 3](https://maven.apache.org)

## Build 
build 
```shell
mvn clean install
```
## Run
```shell
cd .\main-component\
mvn spring-boot:run
```
# REST API
