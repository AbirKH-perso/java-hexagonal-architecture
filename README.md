# kata-bank-account

## Requirements

For building and running the application you need:

- [JDK 1.19](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
- [Maven 3](https://maven.apache.org)

## Build 
build 
```shell
mvn clean install
```
## Run
```shell
cd .\main-component\
mvn spring-boot:run
```
